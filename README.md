# Rafeeq Al-Haramain | رفيق الحرمين

رفيق الحرمين هو تطبيق متكامل ومساند رقمي تفاعلي لضيوف الرحمن من الحجاج والمعتمرين لتقديم الإرشاد الجغرافي والشرعي والميداني أثناء تأدية المناسك.

## المميزات
- **موجّه المناسك**: استعراض مناسك الحج والعمرة خطوة بخطوة مع شرح الأركان والواجبات والمستحبات.
- **خدمة المرشد الميداني**: نظام متكامل للإرشاد والبحث الشرعي والترجمة الفورية للغات متعددة عبر قنوات آمنة وموثوقة مستندة لدار الإفتاء السعودية.
- **الخرائط التفاعلية**: تحديد المرافق والمواقع الخدمية الهامة القريبة في الحرمين الشريفين والمشاعر المقدسة ومطابقتها مع تيسيرات الحالة الصحية للمستخدمين.
- **لمّ الشمل ومشاركة الموقع**: تتبع أفراد الحملة أو العائلة في الحرم لتفادي الضياع ومشاركة الإحداثيات الجغرافية بدقة وأمان.
- **الخدمات المخصصة**: تصنيف الخدمات حسب صفة المستخدم (حاج، معتمر، زائر، متطوع، أو موظف).

## التقنيات المستخدمة
- **الواجهة الأمامية**: React (Vite), TypeScript, Tailwind CSS, Motion (Framer).
- **الواجهة الخلفية**: Node.js (Express) مع ميزات تأمين متطورة وتحديد لمعدل الطلبات لحماية موارد الخادم.
- **قاعدة البيانات**: Google Firebase Firestore (اختياري لتفعيل لمّ الشمل ومشاركة الإحداثيات).

## التشغيل المحلي

1. تثبيت الاعتمادات:
```bash
npm install
```

2. إعداد المتغيرات البيئية (اختياري للخدمات السحابية والإرشاد):
قم بإنشاء ملف `.env` بناءً على `.env.example`.

3. بدء تشغيل الخادم في بيئة التطوير:
```bash
npm run dev
```
